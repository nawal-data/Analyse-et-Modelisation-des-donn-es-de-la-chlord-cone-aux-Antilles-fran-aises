# Analyse-et-Modelisation-des-donn-es-de-la-chlord-cone-aux-Antilles-fran-aises
📌 Projet : Analyse Chlordécone – Notebook R
Auteur : Nawal El Hamdaoui

🎯 Objectif
Analyser les données de contamination à la chlordécone en Martinique pour identifier les zones à risque et comprendre les facteurs explicatifs.

🛠️ Méthodes
Nettoyage (dates, formats, valeurs manquantes)

Imputation (missForest)

Boxplots multivariés

Distance de Mahalanobis

ACP

FAMD

CAH & K-means

Analyse spatiale

📊 Résultats clés
Contamination structurée par :

historique bananier

topographie

type de sol

localisation géographique

4 clusters cohérents

Zones à risque clairement identifiées

🧭 Recommandations
Prioriser les zones historiquement bananières

Cibler les sols andosoliques

Renforcer la surveillance dans les zones à outliers

Utiliser les clusters pour optimiser les prélèvements
